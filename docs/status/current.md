# 当前状态

## 已确定

- 主导航：推荐、热榜、设置。
- 详情支持问题、回答、文章、想法、评论和用户公开资料。
- 所有远端交互保持只读，回复采用单层弹层查看。
- 固定品牌色和显式明暗主题已建立。
- 推荐预加载保留分页游标，列表请求具有刷新代次与加载锁。
- 详情请求会复用同一内容的在途 Future，避免预加载和页面同时重复请求。
- 用户回答与文章已支持独立分页、错误重试和旧请求失效。
- 回答支持横向分页切换与中线触觉反馈；短正文无需额外滚动即可触发评论预加载。
- 正文图片支持同文画廊、双击和双指缩放、缩放后平移、左右翻图与长按操作。
- 下拉刷新复用 Auto Folo 受控维护的状态机，反悔收回边界与 AppBar 对齐。
- 登录页具备仅 Debug 生效的安全检查点，并对缺失结束回调和 12 秒加载超时兜底。
- 工程维护文档已按产品、架构、操作、状态和历史建立知识地图。
- 发布脚本已在临时 Git 仓库完成版本递增、提交和 annotated tag 验证。

## 已知外部风险

- 知乎私有接口和登录流程可能随上游变化，偶发加载失败不一定能由客户端完全消除。
- 登录依赖 WebView Cookie；上游风控或页面结构变化可能影响成功率。

出现问题时应先记录具体页面、网络错误和是否登录，再判断是页面状态机、接口字段
变化还是上游限制。日志不得包含 Cookie、签名或完整登录 URL。

除 `device-acceptance.md` 中的项目外，当前没有已知的真机前阻塞项。
