# 真机验收清单

以下项目依赖 Android 系统、真实显示屏、WebView 或真实知乎账户，无法由桌面
构建和 Widget 测试完整替代。

## 必须真机

- 完成 WebView 登录，验证 `z_c0/d_c0` Cookie 保存、过期与退出。
- 在已登录和未登录状态下分别浏览推荐、热榜、问题、回答、文章、想法与评论。
- 验证知乎实际分页字段、风控响应和签名算法是否仍与上游匹配。
- 验证 60/90/120Hz 设置、回答横滑手感、振动反馈和长列表掉帧情况。
- 检查小屏、异形屏、深色模式和系统字体放大后的导航、弹层与长文排版。
- 点击外部链接和视频，确认系统浏览器选择器可用。
- 保存图片到相册，验证 Android 7–16 对应的媒体权限和 Gal 插件行为。

## 发布前另行确认

- 使用正式 keystore 构建，并核对证书 SHA-256；当前本地 Release APK 是 debug 签名回退。
- 确认新包名 `io.github.xraygit.tritium` 的安装、覆盖安装策略与旧版数据取舍。
- 替换并验收新的图标和启动图资产。
