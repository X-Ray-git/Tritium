# 产品原则

## 定位

Tritium 是专注阅读体验的非官方知乎 Android 客户端。它提供推荐、热榜、问题、
回答、文章、想法、评论和公开用户资料的阅读入口，不试图复刻知乎的完整社区功能。

## 功能边界

- 主导航固定为“推荐 / 热榜 / 设置”，不再用顶部标签重复占用内容空间。
- 允许展示点赞数、评论数、关注者数等公开信息，但不把只读数量伪装成操作按钮。
- 不提供点赞、关注、回复、收藏、私信、发现页或 AI 功能。
- 回复只采用单层弹层查看，避免多级横向串联和复杂会话树。
- 知乎可识别内容尽量应用内打开；外部网站、视频和未支持内容交给系统浏览器。

## 参考工程边界

- Hydrogen 用于理解知乎接口、分页字段和内容模块。
- Auto Folo 用于参考设计语言、文档维护方式和发布流程。
- Tritium 保持独立实现，不直接修改或依赖两个参考工程。
