# 存储与缓存

## Hive Box

- `settings`：主题模式、默认首页、回答/评论排序、刷新率和触觉反馈偏好。
- `user`：登录标记、Cookie 和当前用户信息。
- `cache`：可丢弃的运行时缓存。

存储路径位于系统应用支持目录下的 `hive` 子目录。测试可通过 `pathOverride` 使用
临时路径，不依赖真实设备目录。

## 迁移约定

删除偏好或改变含义时必须提供一次性清理或兼容读取。旧版动态取色和自定义颜色键已
在初始化时删除，品牌色始终由主题代码显式定义。

## 预加载

推荐流预加载必须同时保存条目、`nextUrl` 和 `isEnd`，消费后才能继续正确分页。
详情预加载复用对应 HTTP 模块的在途 Future，避免列表预加载与详情页重复请求。

预加载只用于性能优化，不是持久业务状态；失败不能阻止页面正常发起请求。
